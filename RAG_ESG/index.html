GreenBond-RAG

This repo is meant to be run as a Python app (Streamlit or Tkinter). If you opened Live Server, you're seeing this page instead of a directory listing.

How to run

  • Install deps: pip install -r requirements.txt
  • Streamlit demo: streamlit run src/app_streamlit.py
  • Tkinter app: python -m src.ui.app

If you need a web UI served from this file, replace this content with your desired frontend and reload Live Server.
